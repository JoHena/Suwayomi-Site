# Suwayomi Docs

Suwayomi-Server is a free and open source manga reader server that runs extensions built for [Mihon (Tachiyomi)](https://mihon.app/). It is an independent Mihon (Tachiyomi)-compatible project, not a fork of Mihon (Tachiyomi).

Suwayomi-Server itself has no interface of its own — you access your library through a separate client app (see [Clients](/en/docs/clients)). Once you have a client connected to a running server, it can:

- Install and run Mihon (Tachiyomi) extensions, giving you the same sources
- Search and browse installed sources
- Maintain a library with categories
- Automatically check for and download new chapters
- Let you read manga offline
- Back up and restore your library using Mihon (Tachiyomi)-compatible backups
- Track progress via MyAnimeList, AniList, MangaUpdates, and more
- Bypass Cloudflare challenges via FlareSolverr/Byparr
- Serve your library over OPDS/OPDS-PSE
