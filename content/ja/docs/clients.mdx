# クライアント

Suwayomi-Server 自体にはインターフェースがありません — フロントエンドとなるクライアントアプリが必要です。[Suwayomi-Server を直接ダウンロード](https://github.com/Suwayomi/Suwayomi-Server/releases/latest)すると、[Suwayomi-WebUI](https://github.com/Suwayomi/Suwayomi-WebUI) が同梱されているため、多くの人は別途クライアントをインストールする必要はありません。

Suwayomi-Server 向けの既知のクライアント/ユーザーインターフェースの一覧です。それぞれの現在の機能については、各プロジェクトのリポジトリを確認してください。
