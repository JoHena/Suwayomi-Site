# Suwayomi ドキュメント

Suwayomi-Server は、[Mihon (Tachiyomi)](https://mihon.app/) 用に作られた拡張機能を実行できる、無料でオープンソースのマンガリーダーサーバーです。Mihon (Tachiyomi) と互換性を持つ独立したプロジェクトであり、Mihon (Tachiyomi) のフォークではありません。

Suwayomi-Server 自体には独自のインターフェースがありません — ライブラリには別のクライアントアプリを通してアクセスします([クライアント](/ja/docs/clients)を参照)。稼働中のサーバーにクライアントを接続すると、次のことができます。

- Mihon (Tachiyomi) の拡張機能をインストール・実行し、同じソースを利用できる
- インストール済みのソースを検索・閲覧できる
- カテゴリー分けしたライブラリを管理できる
- 新しいチャプターを自動的にチェック・ダウンロードできる
- オフラインでマンガを読める
- Mihon (Tachiyomi) 互換のバックアップでライブラリをバックアップ・復元できる
- MyAnimeList、AniList、MangaUpdates などで進捗を追跡できる
- FlareSolverr/Byparr で Cloudflare の認証を回避できる
- OPDS/OPDS-PSE でライブラリを配信できる
