# Documentación de Suwayomi

Suwayomi-Server es un servidor lector de manga gratuito y de código abierto que ejecuta extensiones creadas para [Mihon (Tachiyomi)](https://mihon.app/). Es un proyecto independiente compatible con Mihon (Tachiyomi), no un fork de Mihon (Tachiyomi).

Suwayomi-Server en sí no tiene interfaz propia — accedes a tu biblioteca a través de una aplicación cliente independiente (consulta [Clientes](/es/docs/clients)). Una vez que tengas un cliente conectado a un servidor en ejecución, podrás:

- Instalar y ejecutar extensiones de Mihon (Tachiyomi), obteniendo las mismas fuentes
- Buscar y explorar las fuentes instaladas
- Mantener una biblioteca organizada por categorías
- Comprobar y descargar automáticamente los nuevos capítulos
- Leer manga sin conexión
- Hacer copias de seguridad y restaurar tu biblioteca usando copias compatibles con Mihon (Tachiyomi)
- Seguir tu progreso a través de MyAnimeList, AniList, MangaUpdates y más
- Evitar los desafíos de Cloudflare mediante FlareSolverr/Byparr
- Servir tu biblioteca mediante OPDS/OPDS-PSE
