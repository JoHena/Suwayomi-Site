# Clientes

Suwayomi-Server no tiene interfaz propia — necesitas una aplicación cliente como frontend. Si [descargas Suwayomi-Server directamente](https://github.com/Suwayomi/Suwayomi-Server/releases/latest), obtienes una copia incluida de [Suwayomi-WebUI](https://github.com/Suwayomi/Suwayomi-WebUI), por lo que la mayoría de las personas no necesitan instalar un cliente aparte para empezar.

Aquí tienes una lista de los clientes/interfaces de usuario conocidos para Suwayomi-Server. Consulta el repositorio de cada proyecto para ver sus funciones actuales.
