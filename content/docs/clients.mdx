# Clients

Suwayomi-Server has no interface of its own — you need a client app as a front-end for it. If you [download Suwayomi-Server directly](https://github.com/Suwayomi/Suwayomi-Server/releases/latest), you get a bundled copy of [Suwayomi-WebUI](https://github.com/Suwayomi/Suwayomi-WebUI) with it, so most people don't need to install a separate client to get started.

Here's a list of known clients/user interfaces for Suwayomi-Server. Check each project's own repository for its current feature set.
